  • Minecraft Server

    • So when I was in fourth grade, Minecraft was a pretty popular game. All of my fiends played it. We only had one problem which was playing on the same server. Public servers worked, but we had to deal with other people and we didn't have the power to control the server. I eventualy learned how to create a server and made a Vanilla server that we could all play on.

    Eventually my server stopped working. After a while I decided to remake my server. I decided I would create a Bukkit server. This type of server was more interesting due to the fact that it could support plug-ins that could change game play.
